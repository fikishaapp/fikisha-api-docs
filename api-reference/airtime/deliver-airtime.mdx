# Deliver Airtime

Delivers airtime to the specified phone number.

## Request

<ParamField body="phoneNumber" type="string" required>
  The phone number to deliver airtime to with country code (e.g., +254712345678)
</ParamField>

<ParamField body="amount" type="number" required>
  The amount of airtime to deliver (in KES)
</ParamField>

<ParamField body="provider" type="string" required>
  The network provider (e.g., safaricom, airtel, telkom)
</ParamField>

```json
{
  "phoneNumber": "+254712345678",
  "amount": 100,
  "provider": "safaricom"
}
```

## Response

<ResponseField name="success" type="boolean">
  Indicates if the airtime delivery was successful
</ResponseField>

<ResponseField name="transactionId" type="string">
  The unique transaction ID for this airtime delivery
</ResponseField>

<ResponseField name="status" type="string">
  The status of the transaction (completed, pending, failed)
</ResponseField>

```json
{
  "success": true,
  "transactionId": "txn_1234567890",
  "status": "completed",
  "message": "Airtime delivered successfully"
}
```

```python Python
import requests

def deliver_airtime():
    try:
        url = 'https://api.fikisha.com/v1/airtime'
        payload = {
            'phoneNumber': '+254712345678',
            'amount': 100,
            'provider': 'safaricom'
        }
        headers = {
            'Content-Type': 'application/json',
            'X-Vendor-ID': 'YOUR_VENDOR_ID',
            'X-API-Key': 'YOUR_API_KEY'
        }
        
        response = requests.post(url, json=payload, headers=headers)
        response.raise_for_status()
        
        print('Response:', response.json())
        return response.json()
    except requests.exceptions.RequestException as e:
        print('Error:', e)
        raise e
```
